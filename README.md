# Inventory-Sales-Analysis-Python-
End-to-end data analysis project on retail inventory and sales data. Performed data cleaning, transformation, and visualization using Python.
<img width="1126" height="686" alt="Dash" src="https://github.com/user-attachments/assets/68376626-d896-44a5-bf86-13470df3bdea" />

## Dataset
6 CSV files — Sales, Begin Inventory, End Inventory, Purchases, Purchase Price, Vendor Invoice

## Project Structure
- Data_Cleaning.ipynb — Null handling, duplicate detection, date conversion, string formatting
- Data_Transformation.ipynb — KPI calculations, groupby aggregations, inventory comparison
- Viz.ipynb — Charts and visual insights

## Key Insights
- Identified top 5 cities with maximum stock using line chart
- Found top 5 highest inventory value products using bar chart
- Analyzed average inventory value per store using pie chart
- Compared purchase vs sales price across all products

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn

## Conclusion

This project provided a comprehensive view of inventory and sales
performance across multiple stores, cities, and product categories.

Key Takeaways:
- Certain cities hold significantly higher stock levels than others,
  indicating uneven inventory distribution
- A small set of products contribute to the majority of total
  inventory value
- Purchase prices and sales prices show a strong positive correlation,
  confirming consistent profit margins across products
- Beginning vs ending inventory comparison revealed the net change
  in stock value over the period
- Some stores carry disproportionately high average inventory value,
  which may signal overstocking

This analysis can help business teams make smarter decisions around
restocking, vendor negotiations, and store-level inventory planning.
